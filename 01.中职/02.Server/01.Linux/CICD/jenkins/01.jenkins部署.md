## 一、安装 Java 21

优先建议用 **Java 21**。
```
sudo dnf update -y
sudo dnf install -y fontconfig java-21-openjdk
```
检查版本：
```
java --version
```

## 二、导入 Jenkins 新仓库密钥并添加仓库

从 2026 年 1 月开始，Jenkins Linux 包使用了新的签名密钥；另外 RPM 仓库地址也已经迁移。

执行：
```
sudo rpm --import https://pkg.jenkins.io/rpm-stable/jenkins.io-2026.key
```

创建仓库文件：
```
sudo rpm --import https://pkg.jenkins.io/rpm-stable/jenkins.io-2026.key
```

刷新缓存：
```
dnf makecache
```

三、安装 Jenkins
```
sudo dnf install -y jenkins
```

安装完成后，可以查看包版本：
```
rpm -qi jenkins
```

四、启动并设置开机自启
```
sudo systemctl daemon-reload  
sudo systemctl enable --now jenkins  
sudo systemctl status jenkins
```

## 五、放行防火墙端口

如果系统启用了 firewalld：
```
sudo firewall-cmd --permanent --add-port=8080/tcp
sudo firewall-cmd --reload
```

然后浏览器访问:
```
http://服务器IP:8080
```

## 六、获取初始化管理员密码

首次进入 Web 页面时，需要输入解锁密码：
```
sudo cat /var/lib/jenkins/secrets/initialAdminPassword
```

## 七、首次初始化建议

进入页面后建议这样选：

1. 选择 **Install suggested plugins**,安装插件需要翻墙，否则部分插件无法安装
2. 创建管理员账号
3. 配置 Jenkins URL
4. 进入控制台
